# SkyShell

A simple shell for a university project.

## Usage

`make && ./skyshell`

## Features

- [x] Basic commands
- [x] Change directory (cd)
  - [x] Home directory (~) or (NULL)
  - [x] Last directory (-)
- [x] Pipes
- [x] Background processes
- [x] Read in double quotes (") -> `echo "Hello World"`
  - Single quotes (') are not supported
